# b

## Permanent Game Hub

Use this standalone hub page to always launch the game from one place and swap/update build URLs without editing code.

- Hub file in repo: `GameHub.html`

### Live links

- Dev branch hub (this branch):
  - `https://cdn.jsdelivr.net/gh/jf609656-svg/b@cursor%2Fgame-code-expansion-921c/GameHub.html`
- Main hub (after merge to main):
  - `https://cdn.jsdelivr.net/gh/jf609656-svg/b@main/GameHub.html`

### How to use

1. Open the hub URL once.
2. Pick a channel (`Dev Branch`, `Main`, or `Custom URL`).
3. Click **Save as Default**.
4. From then on, just open the same hub link and click **Launch in Hub** or **Reload Game**.
